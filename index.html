Project Overview

Our project investigates the development of a direct methanol fuel cell (DMFC) system, capable of charging a portable battery pack using methanol as a chemical energy source. The project includes electrical, chemical, and mechanical integration to create a robust, testable prototype.

The system uses a 10-cell DMFC stack operating on 1M methanol feed solution, with performance monitored through custom Arduino circuitry and sensors. We’ve also developed our own peristaltic calibration curve, explored MeOH crossover behavior, and built active control systems for pump regulation.

Build and Run Instructions

Materials & Components

  • DMFC 10-cell stack (Fuel Cell Store Flex-Stak)
  • 1M methanol-water solution (3% by weight MeOH)
  • Arduino Uno, breadboard, resistors, MOSFET
  • Peristaltic pump (3x5V configuration)
  • Flow meters, voltage/current sensors, and optional fan

Setup Procedure

  1. Hydrate stack by flowing DI water for 15–20 mins.
  2. Switch to 1M MeOH feed at 100 cc/min (adjust if needed).
  3. Power up circuit and monitor real-time voltage.
  4. Connect Arduino for data logging.
  5. Use included code (see GitHub repo) for plotting power output.

Experimental Data

  • Peristaltic Pump Calibration Curve
  • Fuel Cell I/V and Power Output Tables

Under conservative operating conditions (50 mA/cm²), we achieved ~19W across the 10-cell stack. With higher current densities (~100 mA/cm²), outputs approached 30W. Effluent concentration was verified at ~10%.
